# Heap Animation

This Manim animation visualizes heap insertion and extraction operations.

## Video Output

The rendered 1080p60 animation will be available as `heap_animation.mp4` after rendering.

## Installation

First, install Manim:

```bash
pip install manim
```

## Running the Animation

```bash
# Low quality (fast preview)
manim -pql heap_animation.py HeapVisualization

# High quality (for presentations)
manim -pqh heap_animation.py HeapVisualization

# 4K quality
manim -pqk heap_animation.py HeapVisualization
```
