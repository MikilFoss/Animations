# Binary Tree Animation

This Manim animation visualizes the structure and properties of binary trees.

## Video Output

The rendered 1080p60 animation will be available as `binary_tree_animation.mp4` after rendering.

## Installation

First, install Manim:

```bash
pip install manim
```

## Running the Animation

```bash
# Low quality (fast preview)
manim -pql binary_tree_animation.py BinaryTreeVisualization

# High quality (for presentations)
manim -pqh binary_tree_animation.py BinaryTreeVisualization

# 4K quality
manim -pqk binary_tree_animation.py BinaryTreeVisualization
```
