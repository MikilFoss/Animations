# Red-Black Tree Animation

This Manim animation visualizes Red-Black tree insertion with rotations.

## Video Output

The rendered 1080p60 animation will be available as `redblack_tree_animation.mp4` after rendering.

## Installation

First, install Manim:

```bash
pip install manim
```

## Running the Animation

```bash
# Low quality (fast preview)
manim -pql redblack_tree_animation.py RedBlackTreeVisualization

# High quality (for presentations)
manim -pqh redblack_tree_animation.py RedBlackTreeVisualization

# 4K quality
manim -pqk redblack_tree_animation.py RedBlackTreeVisualization
```
