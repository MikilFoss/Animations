# Binary Search Tree Animation

This Manim animation visualizes BST insertion and search operations.

## Video Output

The rendered 1080p60 animation will be available as `bst_animation.mp4` after rendering.

## Installation

First, install Manim:

```bash
pip install manim
```

## Running the Animation

```bash
# Low quality (fast preview)
manim -pql bst_animation.py BSTVisualization

# High quality (for presentations)
manim -pqh bst_animation.py BSTVisualization

# 4K quality
manim -pqk bst_animation.py BSTVisualization
```
