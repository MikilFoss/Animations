# AVL Tree Animation

This Manim animation visualizes AVL tree insertion with rotations.

## Video Output

The rendered 1080p60 animation will be available as `avl_tree_animation.mp4` after rendering.

## Installation

First, install Manim:

```bash
pip install manim
```

## Running the Animation

```bash
# Low quality (fast preview)
manim -pql avl_tree_animation.py AVLTreeVisualization

# High quality (for presentations)
manim -pqh avl_tree_animation.py AVLTreeVisualization

# 4K quality
manim -pqk avl_tree_animation.py AVLTreeVisualization
```
